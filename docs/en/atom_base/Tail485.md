## 描述

**Tail485** 是一款为ATOM设计的TTL-RS485转换器，用于TTL电平与RS485电平转换,RS485是一种通信协议标准，用于定义用于串行通信系统的驱动器和接收器的电气特性,支持多点系统，广泛应用于工业领域。当项目设备需要通过RS485进行通信控制时，使用 RS485 Unit 进行接口类型转接会是一个不错的选择。在Tail485的内部集成有DC/DC降压稳压芯片，可以直接将RS485的12V电压转为5V为TypeC接口供电，免去了单独供电的烦恼。
